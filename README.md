# TrGu Mod

A content expansion mod for **Balatro**, created with the **Steamodded / SMODS** framework.

TrGu Mod adds a large collection of original Jokers, Admin cards, enhancements, decks, tags, booster packs, a custom Boss Blind, new audio, custom UI elements, and full English and Russian localization.

> **Current version:** v0.7 Pre-release  
> **Last update:** July 16, 2026

---

## Features

The current version of TrGu Mod includes:

- **64 new Jokers**
- **12 Admin cards**
- **6 custom Boss Blinds**
- **2 custom enhancements**
- **2 new decks**
- **2 new tags**
- **4 booster packs**
- **English and Russian localization**

---

## Requirements

TrGu Mod requires:

- **Balatro**
- **Lovely**
- **Steamodded / SMODS**

Use versions of Lovely and Steamodded that are compatible with your installed version of Balatro.

---

## Installation

1. Install **Lovely**.
2. Install **Steamodded / SMODS**.
3. Download the latest TrGu Mod release.
4. Extract the mod folder into your Balatro `Mods` directory.
5. Make sure `main.lua` is located directly inside the TrGu Mod folder and is not nested inside an additional folder.
6. Launch Balatro.

The final folder structure should look similar to this:

```text
Balatro/
└── Mods/
    └── TrGu/
        ├── main.lua
        ├── adminscards.lua
        ├── bossblinds.lua
        ├── deckstags.lua
        ├── editionsenhances.lua
        ├── lang_helpers.lua
        ├── mod_info.lua
        └── assets/
```

The exact folder contents may differ between releases.

## Localization

TrGu Mod currently supports:

- English
- Russian

The mod automatically uses the selected in-game language where localization is available.

## Configuration

The mod includes configurable texture variants for Admin cards.

Open the mod configuration menu through Steamodded to select the available Admin texture style.
